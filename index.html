# Network Quest: The Race to Root Access This folder contains a ready-to-deploy setup for hosting the interactive networking quiz game online. --- ## 📦 Folder Structure ``` network-quest/ ├── index.html ├── README.md └── assets/ ├── sounds/ │ ├── correct.ogg │ ├── wrong.ogg │ └── levelup.ogg └── favicon.ico ``` --- ## 🚀 Hosting on GitHub Pages 1. **Create a new GitHub repository** named `network-quest` (or any name you like). 2. Upload the contents of this folder to your repository. 3. Go to your repository’s settings → **Pages** → set the source to **main branch** and root `/`. 4. Wait about 30 seconds, then your game will be live at: ``` https://yourusername.github.io/network-quest/ ``` --- ## 💻 index.html ```html Network Quest: The Race to Root Access
